# Video Player

A personal Electron-based video player for Windows, built with Apple-minimal glassmorphism aesthetics. Addresses specific pain points with mainstream players: instant speed control, configurable jump seconds, and a clean UI that doesn't hide common actions behind nested menus.

## Features

- **Speed control** — first-level button, no nested menus. Speeds are fully configurable
- **Jump forward/backward** — keyboard arrows jump a configurable number of seconds
- **Screenshot & crop** — press `S` or click the camera button, drag to select a region, it copies to clipboard at native video resolution instantly
- **Settings panel** — edit speed levels, jump seconds, and default volume without touching a config file
- **Recent files** — shows up to 8 recently played files on the empty state for quick access
- **Playback memory** — remembers the last position per file across sessions
- **Drag & drop** — drop a video file anywhere onto the window
- **Frameless window** — custom titlebar with minimize / maximize / close
- **Fullscreen** — `F` key or the fullscreen button; controls auto-hide after 3 s of inactivity
- **Glass overlay** — titlebar and controls float over the video with `backdrop-filter` blur

### Keyboard shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `←` / `→` | Jump backward / forward |
| `F` | Toggle fullscreen |
| `S` | Screenshot crop mode |
| `Ctrl+O` | Open file dialog |
| `Esc` | Cancel crop |

### Supported formats

`mp4`, `webm`, `mov`, `avi`, `mkv` (anything Chromium can decode natively)

## Getting started

```bash
npm install
npm start          # launch in dev mode
npm test           # run Jest unit tests
npm run build      # package to dist/ (Windows NSIS installer)
```

Requires Node.js 18+ and npm.

## Configuration

`config.json` at the project root is read on startup. Edit it directly or use the in-app settings panel (gear icon).

```json
{
  "speeds": [0.75, 1, 1.25, 1.5, 2],
  "jumpSeconds": 15,
  "defaultVolume": 70
}
```

| Field | Type | Description |
|-------|------|-------------|
| `speeds` | `number[]` | Speed options shown in the speed menu |
| `jumpSeconds` | `number` | Seconds to skip on arrow key / jump button |
| `defaultVolume` | `number` | Initial volume 0–100 |

Changes made via the settings panel are written back to `config.json` and applied immediately.

## Project structure

```
video-player/
├── main.js           # Electron main process — BrowserWindow, IPC handlers
├── preload.js        # Context bridge — exposes safe IPC API to renderer
├── config.json       # User config (speeds, jumpSeconds, defaultVolume)
├── assets/
│   └── icon.png      # App icon (generated by scripts/gen-icon.js)
├── renderer/
│   ├── index.html    # App shell + all DOM structure
│   ├── style.css     # All styles — Apple minimal, CSS variables, glassmorphism
│   ├── utils.js      # formatTime(), clamp() — dual export (browser + Jest)
│   └── player.js     # All renderer logic — playback, UI state, IPC calls
├── scripts/
│   └── gen-icon.js   # Generates assets/icon.png via pngjs (run manually)
└── tests/
    └── utils.test.js # Jest tests for formatTime + clamp
```
